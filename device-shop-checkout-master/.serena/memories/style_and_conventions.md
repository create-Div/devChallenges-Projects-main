Tech stack: TypeScript, Vite, plain HTML, CSS, static assets. Module type is ESM. TypeScript is strict with noUnusedLocals, noUnusedParameters, noFallthroughCasesInSwitch, and noEmit. Formatting/linting uses oxfmt and oxlint with default config plus JSX/React-related settings available. Conventions visible in the repo: semantic class names, kebab-case CSS classes, static markup in index.html, assets referenced from /public, and modern CSS features (cascade layers, nesting, logical properties, color-mix, oklch, view transitions).
