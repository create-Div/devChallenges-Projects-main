Development commands for this project:\n- npm install — install dependencies (package-lock.json is present)\n- npm run dev — start Vite dev server\n- npm run build — type-check with tsc and build with Vite\n- npm run preview — preview the production build\n- npm run lint — run oxlint on src/ with warnings denied\n- npm run lint:fix — auto-fix lint issues in src/\n- npm run format — format src/ and index.html with oxfmt\n- npm run format:check — check formatting without modifying files\nUseful Linux utilities available: git, ls, grep, find, pwd, mv, cp, rm.
